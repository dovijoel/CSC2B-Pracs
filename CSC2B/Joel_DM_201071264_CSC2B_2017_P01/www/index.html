Welcome to Dovi's simple web server!

This is a very simple web server implemented in Java, with the ability to:
  • Load simple html pages
  • Load images in-line
    • This image was released into the public domain
  • Send binary files - download
    • Right-click and choose "Save link as..." to download. PDF doesn't render in Chrome, but works in Edge. Hmm...
  • Serve content from a custom directory, specified from the command line (bonus points :D)
  • Sub directory dynamically works!
  • Send appropriate HTTP response codes - 200, 404 or 500.
    • The error 500 code can be tested as in this link
